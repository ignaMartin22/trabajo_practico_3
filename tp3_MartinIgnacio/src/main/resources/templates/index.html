Primer Sección

Java

Java es una plataforma informática de lenguaje de programación.
Ha evolucionado desde sus humildes comienzos hasta impulsar una gran parte del mundo digital actual, ya que es una plataforma fiable en la que se crean muchos servicios y aplicaciones.
Los nuevos e innovadores productos y servicios digitales diseñados para el futuro también siguen basándose en Java.

Características de Java:

  • Orientado a Objetos
  • Portabilidad
  • Robusto y seguro

Características de Java

Características Descripción
Orientado a Objetos Sigue los principios de la programación orientada a objeto (encapsulamiento, herencia, polimorfismo, abstracción)
Portabilidad Utiliza la máquina virtual de Java (JVM) que actúa como capa de abstracción entre el programa Java y el SO subyacente
Robusto y seguro Posee un sistema de manejo de excepciones que permite a los programadores detectar y manejar errores de manera estructurada

Segunda Sección

HTML y CSS

HTML (Hypertext Markup Language): Es el lenguaje estándar para crear y estructurar páginas web,
utilizando etiquetas para definir el contenido y la organización de una página.

CSS (Cascading Style Sheets): Es el lenguaje utilizado para controlar el estilo y la presentación visual de los elementos
de una página web, incluyendo el diseño, el color, la fuente y el formato.

Características de html:

  • Estructura basada en etiquetas
  • Lenguaje descriptivo
  • Lenguaje de Marcado

Características de css

Características Descripción
Jerarquía y cascada Utiliza un sistema de cascada y herencia para aplicar el estilo a los elementos
Compatibilidad multiplataforma Es un estándar web, lo que garantiza su compatibilidad y consistencia en diferentes navegadores
Reutilización de estilos Permite agrupar y reutilizar estilos a través de selectores, clases e id´s

Tercer Sección

Bootstrap

Bootstrap es un framework de código abierto front-end para desarrollar sitios web y aplicaciones web de forma rápida y responsiva.

Características de Bootstrap

  • Diseño responsive
  • Personalización
  • Compatibilidad multplataforma
Características Descripción
Diseño Responsive Proporciona un sistema de cuadrícula y herramientas de diseño que se adaptan automáticamente a diferentes pantallas
Personalización Los estilos de Bootstrap se pueden personalizar fácilmente a través de variables Sass y temas personalizados.
Compatibilidad Multiplataforma Bootstrap es compatible con los principales navegadores web y dispositivos móviles.

Requisitos de cursada

  • Haber aprobado PE y ED
  • Minimo 80% de asistencia a teoria y práctica
  • Tres exámenes parciales a lo largo del cuatrimestre (nota minima: 6)
  • Se debe obtener un promedio general de 7 para aprobar la materia
